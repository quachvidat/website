== Description ==

The Easy Login Form is a Wordpress plugin that allows you to change the styling of the login form.

Using the Easy Login Form is really easy, just follow this steps:

1. Go to Settings -> Easy Login Form
2. Specify your website logo location (inside your theme). For example: images/logo.png
3. Set the Background page colour. Using the colour picker! you don't need to know the colour code.
4. Set the Background form colour, and the Front form colour.
5. Press "save changes".
6. Check your result on "Preview", and if you liked it, that's it!

Enjoy! :)

*Thank you for downloading! your feedback is well appreciated!*

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload plugin folder `easy-login-form` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==

= I cannot see the preview =

You have to press "save changes" before.

= I only see black in the colour picker =

You may have to scroll up the side bar of the color picker to show lighter colours.
